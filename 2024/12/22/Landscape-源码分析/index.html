еҸ‘иЎЁдәҺ 2024-12-22 жӣҙж–°дәҺ 2025-03-05 еҲҶзұ»дәҺ Python жң¬ж–Үеӯ—ж•°пјҡ 714 йҳ…иҜ»ж—¶й•ҝ ≈ 1 еҲҶй’ҹ

жҰӮиҝ°

Landscape жҳҜдёҖдёӘйӣҶзҫӨзӣ‘жҺ§з®ЎзҗҶиҪҜд»¶пјҢе®ғеҸҜд»ҘдҪңдёә web жңҚеҠЎжқҘдҪҝз”ЁжҲ–иҖ…зӣҙжҺҘдҪҝз”Ё APIгҖӮLandscape Server еҸҜд»ҘйҖҡиҝҮ Canonical дҪңдёәжүҳз®Ўи§ЈеҶіж–№жЎҲжҲ–иҖ… Software-as-a-Server жЁЎеһӢпјҢд№ҹеҸҜд»ҘиҮӘиЎҢжүҳз®ЎгҖӮLandscape Client е®үиЈ…еңЁ Ubuntu дёҠпјҢе№¶еҗ‘ Server жіЁеҶҢгҖӮ
Landscape иҮӘеҠЁеҢ–дәҶж•ҙдёӘ Ubuntu зі»з»ҹзҡ„е®үе…ЁиЎҘдёҒгҖҒе®Ўи®ЎгҖҒи®ҝй—®з®ЎзҗҶе’ҢеҗҲи§„жҖ§д»»еҠЎгҖӮ

Landscape жәҗд»Јз ҒдҪҝз”ЁдәҶ Twisted дәӢд»¶й©ұеҠЁзҪ‘з»ңеј•ж“ҺгҖӮ

д»Јз Ғз»“жһ„

жҰӮжӢ¬

еңЁ landscape еҢ…зҡ„init.pyдёӯжҲ‘д»¬зңӢеҲ°е®ҡд№үдәҶдёҖдәӣеҸҳйҮҸпјҢиҝҷдәӣеҸҳйҮҸеҮ д№ҺйғҪе’ҢзүҲжң¬еҸ·жңүе…іпјҢиҝҷдәӣеҸҳйҮҸзұ»дјјдәҺ C иҜӯиЁҖдёӯзҡ„е…ЁеұҖе®Ҹе®ҡд№үгҖӮ

иҝҗиЎҢжЁЎејҸ

Landscape-client еҸҜд»Ҙrootз”ЁжҲ·иҝҗиЎҢпјҢиҝӣиҖҢеҸҜд»ҘдҪҝз”ЁдёҖдәӣй«ҳзә§з®ЎзҗҶзү№жҖ§пјҢеҰӮжһңдёҚйңҖиҰҒдҪҝз”Ёиҝҷз§Қзү№жҖ§ж—¶пјҢеҸҜд»ҘдҪҝз”Ёlandscapeз”ЁжҲ·иҝҗиЎҢгҖӮйҖҡиҝҮдҝ®ж”№/etc/default/landscape-clientе’Ң/etc/landscape/client.confж–Үд»¶еҸҜд»ҘеҠһеҲ°пјҲе…·дҪ“з»ҶиҠӮи§ҒйЎ№зӣ®д»Јз ҒREADMEпјүгҖӮ

watchdog жЁЎеқ—

д»Јз ҒдҪҚзҪ®пјҡlandscape/client/watchdog.py

watchdog жҳҜжңҖй«ҳеұӮзҡ„д»Јз ҒпјҢе®ғжё…зҗҶиҝҗиЎҢзҺҜеўғпјҢиҪҪе…Ҙй…ҚзҪ®ж–Үд»¶пјҢз„¶еҗҺеҗҜеҠЁ ractor еҫӘзҺҜгҖӮ

a. clean_environment()пјҡиҜҘеҮҪж•°жё…йҷӨдёҖдәӣеҚұйҷ©е’Ңж•Ҹж„ҹзҡ„еҸҳйҮҸгҖӮ
b. иҪҪе…Ҙй…ҚзҪ®ж–Үд»¶ : е’Ңй…ҚзҪ®ж–Үд»¶зӣёе…ізҡ„е·ҘдҪңиў«жҠҪиұЎдёәдёҖзі»еҲ—е…·жңүз»§жүҝе…ізі»зҡ„зұ»пјҢжңүдәӣзұ»еңЁдёҚеҗҢзҡ„жЁЎеқ—дёӯиў«йҮҚз”ЁгҖӮ

WatchDogService

WatchDogServiceз»§жүҝиҮӘtwisted.serviceпјҢе®һдҫӢеҢ–ж—¶дј йҖ’WatchDogConfigurationй…ҚзҪ®еҸӮж•°еҲ°з§Ғжңүе®һдҫӢеҸҳйҮҸ_configгҖӮ

a. resourceеҢ…жқҘиҮӘдәҺж ҮеҮҶеә“пјҢз”ЁжқҘжөӢиҜ•е’ҢжҺ§еҲ¶зі»з»ҹиө„жәҗзҡ„еҲҶй…ҚгҖӮsoft limitдёҺhard limitзҡ„еҢәеҲ«пјҢеҸӘжңүrootиҝӣзЁӢеҸҜд»ҘжҸҗеҚҮhard limitпјҲеҸӮиҖғLinux/Unixзі»з»ҹзј–зЁӢжүӢеҶҢпјүгҖӮ иө„жәҗзұ»еһӢеҢ…жӢ¬ CPU иҝҗиЎҢж—¶й—ҙгҖҒеҸҜд»Ҙз”ҹжҲҗж–Үд»¶зҡ„жңҖеӨ§з©әй—ҙгҖҒиҝӣзЁӢе Ҷ/ж Ҳзҡ„еӨ§е°ҸгҖҒжү“ејҖж–Үд»¶жҸҸиҝ°з¬Ұзҡ„ж•°йҮҸпјҲRLIMIT_NOFILEпјүзӯүзӯүгҖӮ
b. WatchDogServiceзҡ„startServiceж–№жі•йҰ–е…Ҳз”ҹжҲҗBootStrapListдёӯзҡ„ж–Үд»¶жҲ–иҖ…зӣ®еҪ•пјҢз„¶еҗҺи®ҫзҪ®иө„жәҗйҷҗеҲ¶пјҢжңҖеҗҺз”ҹжҲҗе·Із»ҸcallbackжҝҖеҸ‘зҡ„DeferedеҜ№иұЎпјҢ
иҜҘеҜ№иұЎеўһеҠ зҡ„еӣһи°ғй“ҫ:

watchdog.check_running() | past
start_if_not_running() | die()

watchdogжҳҜеңЁWatchDogServiceеҲқе§ӢеҢ–ж—¶WatchDogзұ»зҡ„е®һдҫӢеҜ№иұЎгҖӮ

start_if_not_runningж–№жі•дјҡи°ғз”Ё_daemonize()ж–№жі•пјҢиҜҘж–№жі•ж №жҚ®е®һйҷ…еҸӮж•°еҲӨж–ӯиҮӘе·ұжҳҜеҗҰиҰҒжҲҗдёәдёҖдёӘdaemonиҝӣзЁӢпјҢеҰӮжһңжҳҜзҡ„иҜқпјҢе°ұи°ғз”Ёdaemonize functionпјҢиҜҘеҮҪж•°дјҡи°ғз”Ё os зҡ„еә•еұӮ APIпјҢи°ғз”ЁдёӨж¬Ўfork()пјҢе№¶и®©зҲ¶иҝӣзЁӢиҮӘжқҖпјҢе…¶д»–е…·дҪ“з»ҶиҠӮеҸҜеҸӮиҖғ Linux/Unixзі»з»ҹзј–зЁӢжүӢеҶҢгҖӮ жңҖеҗҺи°ғз”Ёwatchdogзҡ„startж–№жі•пјҢе®ғж №жҚ®жқЎд»¶еҗҜеҠЁе…¶д»–daemonsе№¶иҝӣиЎҢзӣ‘жҺ§гҖӮ

e. stopServiceж–№жі•
е®ғдё»иҰҒе·ҘдҪңжҳҜи°ғз”Ёwatchdogзҡ„request_exitж–№жі•пјҢиҜҘж–№жі•дјҡеҒҡжҜ”иҫғзІҫз»ҶеӨҚжқӮзҡ„е·ҘдҪңжқҘе°Ҹеҝғең°и®©иҮӘе·ұе’Ңе…¶д»–daemonsз»ҲжӯўпјҢжҜ”еҰӮйҒҝе…ҚйҮҚеҗҜгҖӮ

f. дёҚз®ЎжҳҜstartServiceиҝҳжҳҜstopServiceж–№жі•иҝ”еӣһзҡ„йғҪжҳҜDeferedеҜ№иұЎжҲ–иҖ…DeferedListеҜ№иұЎпјҢиҝҷдәӣеҜ№иұЎдјҡиў«TwistedеҶ…йғЁдҪҝз”ЁгҖӮ

WatchDog зұ»

WatchDogзұ»иҙҹиҙЈеҗҜеҠЁе…¶д»–жүҖжңүзҡ„daemons(еҢ…жӢ¬brokerпјҢMonitorпјҢManagerзӯү)е№¶зЎ®дҝқе®ғд»¬иҝҗиЎҢгҖӮиҝҷдәӣdaemonsиҝҗиЎҢеңЁдёҚеҗҢзҡ„ractorеҫӘзҺҜдёӯпјҢдҪҝз”ЁAMPеҚҸи®®йҖҡдҝЎгҖӮ landscape_reactorжҳҜEventHandlingReactorзұ»зҡ„е®һдҫӢеҜ№иұЎгҖӮ

a. EventHandlingReactorзұ»е°ҒиЈ…дәҶtwisted.internet.reactorпјҢз»§жүҝиҮӘEventHandlingReactorMixinзұ»пјҢеўһеҠ дәӢд»¶еӨ„зҗҶзү№жҖ§гҖӮ е…¶дёӯcallInThreadжқҘжәҗдәҺTwistedеҜ№дәҺеӨҡзәҝзЁӢзҡ„ж”ҜжҢҒгҖӮTwistedжӣҙеӨҡеә”з”ЁдәҺ I/O еҜҶйӣҶеһӢеә”з”ЁпјҢеҚід»»еҠЎзҡ„и®Ўз®—д»»еҠЎдёҚдјҡжҢҒз»ӯеҫҲй•ҝж—¶й—ҙпјҢдҪҶжҳҜеҰӮжһңжңүдёҖдёӘ
д»»еҠЎеҫҲиҖ—ж—¶зҡ„иҜқпјҢTwistedзҡ„ractorеҫӘзҺҜжӯӨж—¶е®Ңе…Ёиў«иҜҘд»»еҠЎеҚ з”ЁпјҢе…¶д»–д»»еҠЎдјҡйқўдёҙйҘҘйҘҝзҡ„й—®йўҳгҖӮеӣ жӯӨе°ҶиҜҘд»»еҠЎж”ҫеҲ°иҫ…еҠ©зәҝзЁӢдёӯеҺ»жү§иЎҢпјҢreactorдё»еҫӘзҺҜеҸҜд»Ҙеҫ—еҲ°и§Јж”ҫгҖӮcallFromThreadдёӯзҡ„functionеҸҜд»Ҙз«ӢеҚіеңЁreactorзәҝзЁӢдёӯжү§иЎҢпјҢдҪҶдёҚдҝқиҜҒзәҝзЁӢе®үе…ЁпјҢеҚіreactorз«ӢеҚіиў«еҲҮжҚўпјҹеҰӮжһңе®ғжӯЈеңЁжү§иЎҢд»»еҠЎеҰӮдҪ•пјҹгҖӮеҰӮжһңиҜҘ functionдёҖзӣҙеңЁиҝҗиЎҢпјҢreactorиў«йҳ»еЎһпјҢе…¶д»–д»»еҠЎдјҡйқўдёҙйҘҘйҘҝй—®йўҳгҖӮеҸӮзңӢй“ҫжҺҘ2е’Ң3гҖӮ

b. EventHandingReactorMixinзұ»иҙҹиҙЈжҝҖжҙ»еңЁиў«ж ҮиҜҶдәӢд»¶дёҠз»‘е®ҡзҡ„handlersгҖӮйҮҚиҰҒзҡ„дёҖзӮ№жҳҜиҝҷдәӣдәӢд»¶зҡ„еӨ„зҗҶжҳҜеҗҢжӯҘзҡ„иҝҳжҳҜејӮжӯҘзҡ„пјҹ call_onж–№жі•жіЁеҶҢдәӢд»¶handlerпјҢдёҖдёӘдәӢд»¶зұ»еһӢеҸҜд»Ҙз»‘е®ҡеӨҡдёӘhandlerпјҢе№¶ж №жҚ®handlerзҡ„дјҳе…Ҳзә§жҺ’еәҸгҖӮ fairж–№жі•ж №жҚ®дјҳе…Ҳзә§йЎәеәҸжҝҖеҸ‘дёҖдёӘдәӢд»¶зұ»еһӢзҡ„жүҖжңүhandlerпјҢе№¶иҝ”еӣһжүҖжңүhandlerзҡ„жү§иЎҢз»“жһңгҖӮ иҝҷдәӣжү§иЎҢз»“жһңжҲ–и®ёжҳҜDeferedпјҢеҚіе®ғзҡ„з»“жһңиҝҳдёҚиғҪз«ӢеҚіеҫ—еҲ°гҖӮ cancel_callж–№жі•еҲ йҷӨдәӢд»¶зҡ„жҹҗдёӘhandlerз»‘е®ҡгҖӮ

c. EventIDзұ»пјҢеҜ№дәҺжҜҸдёӘhandlerеҸӘеҜ№еә”дёҖз§ҚдәӢд»¶зұ»еһӢпјҢеӣ жӯӨпјҲevent_typeпјҢ (handlerпјҢ priority)пјүеҸҜд»Ҙе”ҜдёҖж ҮиҜҶдёҖдёӘhandlerгҖӮ

d. WatchDogжіЁеҶҢдәҶSIGUSR1дҝЎеҸ·зҡ„handlerпјҢиҜҘhandlerдјҡйҖҡзҹҘжүҖжңүdaemonsи°ғз”Ёrotate_logs()гҖӮ

e. check_runningж–№жі•жЈҖжөӢе“ӘдәӣdaemonsжӯЈеңЁиҝҗиЎҢпјҢз»“жһңж”ҫеңЁдёҖдёӘListдёӯпјҢдҪҶз»“жһңжң¬иә«жҳҜдёҖдёӘDeferedеҜ№иұЎгҖӮgather_resultsе°ҶжүҖжңүDeferedsж”ҫеҲ°DeferedListеҜ№иұЎдёӯз®ЎзҗҶгҖӮDeferdListдјҡзӯүеҫ…жүҖжңүе…¶з®ЎзҗҶзҡ„Deferedsеӣһи°ғй“ҫе®ҢжҲҗеҗҺеҶҚиҝҗиЎҢиҮӘе·ұзҡ„еӣһи°ғй“ҫпјҢиҝҷж ·жӯЈеҘҪе®һзҺ°ејӮжӯҘеҪўејҸзҡ„check_runningж–№жі•гҖӮ

f. startж–№жі•еҗҜеҠЁжүҖжңүе…¶д»–зҡ„daemonsпјҢе№¶и°ғз”Ёstart_minitoringзӣ‘жҺ§е®ғд»¬зҡ„иҝӣзЁӢзҠ¶жҖҒгҖӮstart_monitoringж–№жі•дјҡеҗҜеҠЁдёҖдёӘеӣһи°ғ_checkж–№жі•пјҢ_checkж–№жі•еңЁе®ҢжҲҗдёҖж¬ЎcheckеҗҺеўһеҠ иҮӘе·ұеҲ°reactorеӣһи°ғй“ҫдёӯпјҢе»¶ж—¶ 5 з§’гҖӮеҪ“жЈҖжөӢеҲ°жҹҗдёӘdaemonеңЁ 5 ж¬ЎжҺўжөӢдёӯеқҮжІЎжңүеӣһеӨҚпјҢWatchDogеҲӨж–ӯиҜҘdaemonе·Із»Ҹжӯ»жҺүпјҢжүҖд»ҘжҺҘдёӢжқҘе°ұйҮҚеҗҜе®ғпјҢе№¶жё…йӣ¶жҺўжөӢеӨұиҙҘи®Ўж•°еҷЁгҖӮ
